# Flash Tickets — Frontend

React 19 + Vite + Tailwind v4 client. See the [root README](../README.md) for
the full project overview, architecture, and setup instructions.

```bash
npm install
cp .env.example .env   # VITE_CLERK_PUBLISHABLE_KEY, VITE_BACKEND_URL
npm run dev
```

## Scripts

- `npm run dev` — start the Vite dev server
- `npm run build` — production build
- `npm run lint` — ESLint
